# Changelog

All notable changes to the LLM Trace Viewer extension will be documented in this file.

## [0.1.1] - 2025-01-03

### Changed
- Rebranded to "LLM Trace Viewer"
- Updated command titles to use "LLM Trace:" prefix
- New extension icon

## [0.1.0] - 2024

### Added
- Initial release
- Connect to Langfuse instances with API keys
- Browse traces with date range filtering
- Search traces by name, user ID, or metadata
- View detailed trace information including inputs, outputs, and timing
- Open traces directly in Langfuse web UI
- Support for multiple connections
- Configurable page size and auto-refresh
